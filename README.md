# Insulinkalkylator – GitHub Pages

Det här repot är färdigt för GitHub Pages med GitHub Actions.

## Publicera

1. Skapa ett nytt repo på GitHub.
2. Ladda upp alla filer i detta paket till repots rot.
3. Commit:a till `main`.
4. Gå till **Settings → Pages**.
5. Under **Build and deployment**, välj **Source: GitHub Actions**.
6. Vänta tills workflowen **Deploy GitHub Pages** har kört klart.

Din sida publiceras sedan på din GitHub Pages-adress.

## Appfunktioner

- Installerbar PWA
- Offline-cache via service worker
- Lokal historik och sparade senaste värden
- Snabbläge för snabbare nya beräkningar

## Inställningar i appen

- Frukost 06:00–10:30: 1 E per 5 g kolhydrater
- Övrig tid: 1 E per 7 g kolhydrater
- Målblodsocker: 5,8 mmol/L
- ISF: 2,0 mmol/L per E
- Avrundning: hela enheter
